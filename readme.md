## **README for Codex**

# Codex - Real-Time Online Code Compiler

## **Overview:**

Codex is a web-based code editor that allows users to write HTML, CSS, and JavaScript in a single interface and see the output instantly on the same page. It simplifies web development and learning by providing an all-in-one platform.

## **Features:**

* Real-time preview of HTML, CSS, and JS code.
* Three-panel editor (HTML, CSS, JS) on the left side.
* Output display on the right side updating instantly.
* No need for external editors or browsers.
* Clean and intuitive interface.

## **Problem Solved:**

Codex removes the hassle of switching between coding environments and browsers, streamlining the development process and making it beginner-friendly.

## **Benefits:**

* Speeds up the development workflow.
* Helps learners visualize code instantly.
* Reduces setup and file management complexity.

## **Technologies Used:**

* HTML
* CSS
* JavaScript

## **How to Use:**

1. Open Codex in your browser.
2. Write HTML, CSS, and JavaScript in the respective panels on the left.
3. View the live output on the right panel.
4. Experiment and iterate without refreshing the browser.

## **Future Enhancements:**

* Add support for saving projects.
* Include frameworks like React or Vue support.
* Add themes for the code editor (dark/light mode).
